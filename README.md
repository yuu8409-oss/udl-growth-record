# UDL 행동성장 기록

초등학교 교사가 특수교육대상학생에게 적용한 UDL 지원 전략과 행동 변화를 실명 없이 기록하는 한국어 웹앱 시제품입니다.

## 사용 방법

1. `index.html`을 브라우저로 엽니다.
2. 학생은 실명 대신 `A학생`, `1-01` 같은 코드로 등록합니다.
3. 관찰 행동의 조작적 정의를 입력합니다.
4. 수업 중 학생, 날짜, 교과, 시간대, UDL 전략, 빈도, 지속시간, 강도, 참여율을 기록합니다.
5. 필요할 때 ABC 기록과 특이사항을 추가합니다.
6. 결과 화면에서 일별 추세, 평균, 변화 양상을 확인합니다.
7. `CSV` 버튼으로 기록을 내보내거나 `보고서` 버튼으로 인쇄합니다.

## 개인정보 주의

학생 이름, 장애명, 진단명, 보호자 연락처 등 직접 식별 정보는 입력하지 않는 것을 권장합니다. 실제 학생 자료를 사용할 때는 학교 개인정보 및 정보보안 지침을 확인하세요.

## 해석 주의

이 앱은 UDL 지원의 인과적 효과를 판정하지 않습니다. 화면 문구도 “관찰 기간에 증가 또는 감소 양상이 나타났습니다”처럼 표현합니다. 수면, 건강, 과제 난이도, 교실 환경, 선행사건, 강화 결과 등 다른 요인을 함께 검토해야 합니다.

## 기술 메모

- 서버 설치 없이 실행되는 정적 PWA 형태입니다.
- 자료는 현재 브라우저의 localStorage에 저장됩니다.
- 다른 기기와 자동 동기화되지 않습니다.
- 정식 배포 전에는 IndexedDB, 인증, 백업, 접근 권한 관리로 확장하는 것이 좋습니다.
